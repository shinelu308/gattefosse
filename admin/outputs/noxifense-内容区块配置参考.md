## 注意事项

1. **保存顺序：** 每编辑完一个区块，点击该区块的「**保存内容**」按钮
2. **区块排序：** 使用 ↑↓ 按钮调整区块显示顺序，应与上表序号一致
3. **图片/文件路径：** 图片 URL 和文件路径需先上传到服务器后填入实际路径
4. **视频链接：** 支持 YouTube 链接，占位符请替换为真实视频 URL
5. **富文本格式：** 支持基础 HTML 排版（标题加粗、列表、换行等），建议保持简洁
